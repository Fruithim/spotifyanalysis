Personal Spotify Listening Statistics Analysis

This personal projects serves to use the capabilities of Python to analyse my own personal Spotify listening habits and explore my most listened to artists, patterns in my daily usage, and many more.
This project uses my 2025 Spotify usage

Python Tools Used
- Python
- Pandas
- Matplotlib
- Json
- Jupyter Notebook

Featuring Analysis such as:
- Top 10 Artists
- Listening Behavious per date/hour of the day
- Usage per day of the week

Discoveries:
- Wide range of artists and songs listened, of many genres and languages
- Listening time spikes during morning and evening hours but stows during lunchtime
- No pattern in usage regardless of break or exam period
